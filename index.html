Available Alternative App Marketplace in EU

What's this about

As Apple announced, "Users based in the European Union with an iPhone have the ability to install apps using alternative app marketplaces" from iOS 17.4. But just a few marketplace is available for now. To reduce the effort to find them out, list in this page.

Marketplace App

Name Official Websites Available Remark
AltStore https://altstore.io/ YES Only AltStore PAL is a format of Alternative Marketplace. AltStore (World) is an old version which have nothing to do with Apple new feature. Download from: https://altstore.io/#Downloads
Setapp https://setapp.com/ NO
Mobivention https://mobivention.com/ YES A B2B app, not available for personal users.

Web Distribution Apps

As Apple announced in Web Distribution In the EU, apps more than 1B downloads/month can distribute app from it's own web page, besides App Store. No app is ready for now.
